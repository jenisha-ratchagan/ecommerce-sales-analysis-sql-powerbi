# E-commerce Sales Analysis using SQL & Power BI

## Project Overview

This project analyzes an online retail sales dataset to understand sales performance, customer behavior, and business insights using SQL and Power BI.

## Tools Used

* MySQL
* SQL
* Power BI

## Analysis Performed

* Data quality checks
* Total revenue analysis
* Top-selling products analysis
* Customer revenue analysis
* Country-wise sales analysis
* Monthly sales trend analysis
* Customer segmentation
* RFM customer analysis

## SQL Concepts Used

* Aggregations
* GROUP BY
* CASE statements
* CTEs
* Window functions
* DENSE_RANK()
* ROW_NUMBER()

## Power BI Dashboard

The dashboard includes:

* Sales overview
* Revenue trends
* Product performance
* Customer analysis
* Customer segmentation
